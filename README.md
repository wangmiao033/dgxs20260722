# 帝国雄狮攻略百科

面向玩家的静态攻略百科站，采用百科型信息架构，包含英雄图鉴、技能说明、玩法攻略、等级开放和活动日历。

## 内容规模

- 151 个攻略词条
- 35 名英雄图鉴
- 78 个技能说明
- 主殿等级开放表
- 限时玩法与开服活动
- 全站搜索、分类筛选、词条收藏和移动端适配

## 使用与部署

直接访问 `index.html` 即可运行，不需要数据库和构建命令。仓库可直接部署到 GitHub Pages、Vercel、Netlify 或任意静态服务器。

`payload/` 目录保存压缩后的单文件站点数据，由首页在浏览器内载入并解压。

资料根据游戏内攻略、等级开放和活动表格整理，实际版本内容以游戏内为准。

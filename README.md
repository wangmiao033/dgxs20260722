# 帝国雄狮攻略百科 · 深度版 V2

面向玩家的静态攻略百科站，采用高信息密度百科结构，并将原始攻略表重组为可直接使用的玩家决策工具。

## V2 新增

- 七日开荒路线：按第1天至第7天整理阶段目标与资料入口
- 英雄配队工具：35名英雄筛选、最多5人阵容草稿、职业与定位关键词覆盖检查
- 每日必做清单：固定时间点、轮换玩法提示、浏览器本地打卡
- 主殿等级查询器：拖动等级查看本级新增、累计开放与下一里程碑
- 开服活动查询：七日礼包折扣计算、新服试炼按天筛选、累充奖励表
- 资源规划：免费/付费来源拆分、常见来源反查
- 全站搜索增强：支持词条、工具页、功能解锁和活动关键词
- 词条详情增强：页内目录、上下篇、资源来源、英雄加入配队

## 内容规模

- 151 个百科词条
- 35 名英雄图鉴
- 78 个技能说明
- 69 条功能解锁记录
- 8 个限时玩法记录
- 5 组开服活动资料

## 技术说明

纯静态 HTML/CSS/JavaScript，不依赖数据库、Node.js 或构建命令。Vercel、Netlify、GitHub Pages 和普通静态服务器均可直接部署。

入口文件：`index.html`

正式站点：`https://dgxs.hnchpower.cn/`

资料根据游戏内攻略表整理。站内路线与覆盖检查属于资料重组和玩家提示，具体开放时间、数值及活动内容以游戏内实际版本为准。
